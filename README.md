## рҹ“Ғ Menu Options

<details>
<summary><b>рҹ”§ 01 вҖ” Install Apache2</b></summary>
<br>
Automatically installs Apache2 using the correct package manager for your platform (<code>apt</code> on Linux, <code>pkg</code> on Termux, or auto-detects XAMPP/WAMP on Windows).
<br><br>
</details>

<details>
<summary><b>в–¶пёҸ 02 / вҸ№пёҸ 03 / рҹ”„ 11 вҖ” Start / Stop / Restart Server</b></summary>
<br>
Control the Apache2 service with a single keypress. Live status (<code>в—Ҹ RUNNING</code> / <code>в—Ҹ STOPPED</code>) is always shown at the top of the menu.
<br><br>
</details>

<details>
<summary><b>рҹ–ҘпёҸ 04 вҖ” Create HTML Page вңЁ</b></summary>
<br>
An interactive wizard walks you through entering a page title, body content, and style preferences вҖ” then auto-generates and saves the HTML file directly to your web root.
<br><br>
</details>

<details>
<summary><b>рҹ“Ө 05 вҖ” Upload File & Directory</b></summary>
<br>
Upload any file or folder to the web root. Filenames containing spaces are automatically renamed (spaces вҶ’ underscores) to ensure they work correctly as URLs.
<br><br>
</details>

<details>
<summary><b>рҹ—‘пёҸ 06 вҖ” Delete File & Directory</b></summary>
<br>
Safely remove files or folders from the web root. Confirmation prompts prevent accidental deletion.
<br><br>
</details>

<details>
<summary><b>рҹ”’ 09 вҖ” Generate Secure Link</b></summary>
<br>
Uses <code>cloudflared</code> to create a temporary public HTTPS tunnel link вҖ” share your local server with anyone on the internet without touching your router or firewall.
<br><br>
</details>

<details>
<summary><b>рҹ“Ҡ 12 вҖ” Analytics Dashboard</b></summary>
<br>
Full dashboard showing:
<ul>
<li>Total link opens, server starts, file uploads & deletions</li>
<li>Tunnel sessions and HTML pages created</li>
<li>Server uptime and last start time</li>
<li>Apache2 and cloudflared installation status</li>
<li>Web root size, folder count, file count</li>
<li>Last 10 events with timestamps</li>
</ul>
Export your analytics to JSON or reset all data from within the dashboard.
<br><br>
</details>
